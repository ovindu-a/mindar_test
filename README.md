## Installation Instructions
1. Clone the repo
2. Install dependencies
```ps
npm install
```
3. Run the server
```ps
npm start
```
4. Open the browser and navigate to `localhost:3000`
5. Enjoy!
6. To stop the server, press `Ctrl+C` in the terminal
